Voicer Help

Voicer General Commands

Scroll Up - Scroll the current page up
Scroll Down - Scroll the current page down
Go Back - Go back to the previous page
Go Forward - Go forward to the previous page
Next tab - Switch to the next tab in tab list
Previous tab - Switch to the previous tab in tab list
New tab + "url you want the new tab to go to." - Open the new tab and navigate to the specified website
Switch tab + "the index of the tab in tab list" - Switch to a tab using the index of it
Close tab - Close the current tab


Voicer Google.com Commands

input + "the text you want to input" - Input search term to google
Find it - Submit the search term and get the result
Next result - Select the next result in result list
Previous result - Select the previous result in result list
Go - Go to the selected result


Voicer Youtube.com Commands

input + "the text you want to input" - Input search term to youtube
Find it - Submit search term to youtube and get the result list
Next video - Select next video in list
Previous video - Select previous video in list
Go - Go to the selected video
Skip - Skip advertisement
